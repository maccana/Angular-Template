Another great idea

The app can enable game play through social network connection. A private or public game can be initiated by any member. The centralised cloud app then pulls all song info from the players devices, creates the grid and distributes back to all players game views. When all players are ready (a time limit will be imposed), denoted by a small green light/music/app icon beside their username, the game initialiser can then start the auto generated playlist.

There is several important factors which need to be decided. These include the following:

  • How many simultaneous players?
  • How to match players of similar music tastes?
  • How long should a game be?
